项目名称：暂无

项目介绍：

这是一个我正在持续开发的2D动作游戏Demo，旨在系统性学习Unity引擎的工作流与2D游戏开发技术。项目已完成核心角色控制、动画系统、瓦片地图搭建和基础敌人AI。
这个项目我参考了Night Run Studio的教程，美术资源用的是itch.io的Tiny Swords免费包，重点是自己理解了状态机的工作机制以及Unity的使用，并且把动画、移动、AI整合成一个可运行的Demo。

已完成功能

角色系统：移动控制、动画状态机（Idle/Run）、摄像机跟随

地图系统：瓦片地图编辑、山体遮罩实现场景深度感

敌人系统：哥布林AI、五种状态的状态机（待机/追击/攻击/向上攻击/向下攻击）

美术资源：来自itch.io的Tiny Swords免费素材包
